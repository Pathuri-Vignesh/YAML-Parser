# YAML-Parser
A custom YAML parser built in Python to read YAML files and convert them into structured data.

Features

Supports key value pairs, nested objects, arrays, strings, numbers, booleans, null values, and multiline text blocks.

How to Run

Run the parser with:
python yaml_parser_starter.py sample.yaml

Sample Input
name: Coding Challenges
challenge:
  name: YAML Parser
  valid: true
Purpose

This project helped me learn lexical analysis, parsing, recursion, and how YAML is used in real software and cybersecurity tools.
